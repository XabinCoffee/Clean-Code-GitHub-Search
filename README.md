# Search GitHub

This sample app is built using these Android development practices and tools:

* Jetpack Compose
* Kotlin Coroutines
* Dependency Injection with Hilt
* MVVM architecture with clean code principles

## Features

* Search for GitHub users.
* View a list of a user's public repositories and their followers.
